-> Convolutional Neural Networks

이미지 인식을 효율적으로 처리할 수 있는 방법이다..

image 에 filter를 거쳐 해당 이미지의 Feature를 쉽게 뽑아낼 수 있다~

예시 - sobel filter

-1 0 1         
-2 0 2       
-1 0 1
